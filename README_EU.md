# OpenClaw App

**OpenClaw App Android aplikazioa da, zure OpenClaw laguntzailearekin hitz egiteko** testu, ahots, irudi eta bideo bidez.

Biltegi hau erabiltzaile ez-teknikoentzat pentsatuta dago:
1. Aplikazioak zer egiten duen ulertzeko.
2. Gizaki gisa behar den minimoa prestatzeko.
3. OpenClaw laguntzaileari APK pertsonalizatua eraikitzeko eskatzeko.

## Zer egin behar duzu zuk (gizakia)
1. Erabaki izena, ikonoa, UI hizkuntza, kolore gaia eta audio hobespenak.
2. Eman repo/fork hau zure OpenClaw laguntzaileari.
3. Eskatu `docs/OPENCLAW_AI_REPLICA.md` jarraitzea eta release APK sortzea.
4. APK instalatu eta probatu.

## Zer egingo du OpenClaw laguntzaileak
- Android ingurunearen konfigurazioa
- Bridge/TTS konfigurazioa
- Marka + gaia + hizkuntza pertsonalizazioa
- Release APK konpilazioa
- Balidazio funtzionala

## Dokumentazio nagusia
