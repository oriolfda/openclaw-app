# OpenClaw App

**OpenClaw App es una app Android para comunicarte con tu asistente OpenClaw** por texto, voz, imágenes y vídeo.

Este repositorio está pensado para que una persona no técnica pueda:
1. Entender qué hace la app.
2. Preparar lo mínimo como humano.
3. Pedir a su asistente OpenClaw que construya y personalice el APK.

## Qué haces tú (humano)
1. Decide nombre, icono, idioma UI, tema de color y preferencias de audio.
2. Pasa este repo/fork a tu asistente OpenClaw.
3. Pídele que siga `docs/OPENCLAW_AI_REPLICA.md` y genere APK release.
4. Instala el APK y prueba.

## Qué hará tu asistente OpenClaw
- Setup toolchain Android
- Setup bridge/TTS
- Personalización de marca + tema + idioma
- Compilación APK release
- Validación funcional
