# OpenClaw App

**OpenClaw App é unha app Android para falar co teu asistente OpenClaw** con texto, voz, imaxes e vídeo.

Este repositorio está pensado para que unha persoa non técnica poida:
1. Entender que fai a app.
2. Preparar o mínimo como humano.
3. Pedirlle ao seu asistente OpenClaw que constrúa e personalice o APK.

## Que fas ti (humano)
1. Decide nome, icona, idioma UI, tema e preferencias de audio.
2. Pásalle este repo/fork ao teu asistente OpenClaw.
3. Pídelle seguir `docs/OPENCLAW_AI_REPLICA.md` e xerar APK release.
4. Instala o APK e proba.

## Que fará o asistente OpenClaw
- Setup Android
- Setup bridge/TTS
- Personalización de marca + tema + idioma
- Compilación APK release
- Validación funcional
